Installation
--------

Prerequisites
^^^^^^^^^^^^^^^^
* python >= 3.5
* Tensorflow
* Keras

Install Deepbiome
^^^^^^^^^^^^^^^^

At the command line:

.. code-block:: bash

    # for python 3.x
    
    $ pip3 install git+https://github.com/Young-won/deepbiome.git
